# TicTacBoom
Tic Tac Boom est un jeu multijoueur en ligne où les joueurs doivent trouver des mots contenant une syllabe donnée… avant que la bombe n’explose 💥  👥 Jusqu’à 20 joueurs 📡 Temps réel avec Socket.IO 📚 Dictionnaire de 336 000 mots 🎯 Timer secret et aléatoire 🧠 Logique entièrement côté serveur
